# 天父的話 365 — God Voice

靈修 App，每日由 Groq AI 生成聖經經文與天父的話。

## 使用方式
1. 部署到 Cloudflare Pages / Netlify
2. 打開網站，輸入 Groq API Key（免費申請：https://console.groq.com）
3. 開始靈修

## 部署到 Cloudflare Pages
1. 將此 repo push 到 GitHub
2. 登入 Cloudflare Pages → Create Project → Connect GitHub
3. Build settings 全部留空（純靜態 HTML）
4. Deploy
